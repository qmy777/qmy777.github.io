Hexo中使用gitalk遇到的问题(持续更新中)

  • 未找到相关的 Issues 进行评论，请联系 XXX 初始化创建 & 登录后，总是会跳回首页
  • Error: Network Error
  • 访问”https://api.github.com/user” 401

未找到相关的 Issues 进行评论，请联系 XXX 初始化创建 & 登录后，总是会跳回首页

原因：

  1. 没有开通gitalk应用
  2. 开通好了之后，主题中没有配置gitalk
  3. 配置好的仓库出现问题
  4. 没有给文章评论进行初始化

解决方案：

  1. 没有开通gitalk应用

    • 先进入OAuth Apps进行创建应用的操作

    • 没有https可以使用这个免费的SSL申请工具FreeSSL

  2. 开通好了之后，主题中没有配置gitalk

    • 以amazing为例，yml中需要配置
    • 以amazing为例，source/js/comment-issue-data.js中需要配置

  1. 配置好的仓库出现问题

    • 配置时，gitalk所指定的仓库地址内容有误，请检查url地址是否有误

  2. 没有给文章评论进行初始化

    • 手动登录下即可

Error: Network Error

原因：

  1. 可能在本地4000的时候访问
  2. 未能正确找到仓库 repo，检查一下你的仓库是否配置正确

解决方案：

  1. 可能在本地4000的时候访问

    • 部署到远程后查看

  2. 未能正确找到仓库 repo，检查一下你的仓库是否配置正确

    • 是不是github仓库的url拼写出错或者字母错误

访问” 401

原因：

  1. 没有使用https

解决方案：

  1. 没有使用https
    • 用免费的SSL FreeSSL
